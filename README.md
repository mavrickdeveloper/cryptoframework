# react-native-crpt-module

Crypterium SDK Wrapper

## Installation

```sh
npm install react-native-crpt-module
```

## Usage

```js
import CrptModule from "react-native-crpt-module";

// ...

const result = await CrptModule.multiply(3, 7);
```
